04 — How It Works
01

Image Upload

User uploads a cervical cell image via the web interface (drag & drop or file picker).

02

Preprocessing

Image is resized to 224×224 px, converted to RGB, and pixel values normalized to [0, 1].

03

Feature Extraction

Both ResNet152 and VGG16 independently extract morphological features from the image.

04

Ensemble Prediction

Softmax outputs are combined via optimized weighted averaging to produce final class probabilities.

05

Results Display

Predicted cell type, confidence scores, risk level, and clinical description are shown instantly.

05 — Training Details
  • Dataset balanced with random oversampling to handle class imbalance
  • Data augmentation: rotation, shift, horizontal flip, zoom
  • Transfer learning from ImageNet pre-trained weights
  • Fine-tuning of top layers for cervical cell domain
  • Early stopping + ReduceLROnPlateau to prevent overfitting
  • Ensemble weights optimized on held-out validation set
06 — Disclaimer & Privacy
